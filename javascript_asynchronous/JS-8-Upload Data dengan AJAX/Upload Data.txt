# Upload Data Dengan AJAX #


- Selain menggunakan JSON, Query Param, Form, kadang kita butuh mengirim file ke Server
- Untuk mengirim file ke Server, kita perlu menggunakan object Form Data, 
  lalu kita perlu menambahkan object File ke dalam FormData tersebut
